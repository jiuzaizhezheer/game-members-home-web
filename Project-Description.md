# 游戏会员之家系统设计与实现

## 项目概述

当今游戏产业发展壮大，参与游戏及周边产品的人数越来越多。游戏产品销售、游戏装备及游戏手办交易已然成为一个成熟的产业，为适应这一产业发展需求，本课题旨在设计一个进行游戏、手办等产品的交易系统，同时提供相关一个爱好者进行交流的平台。

## 基本要求

1. 收集查阅相关资料，了解当前游戏平台网站系统基本功能和业务流程。
2. 熟悉并提高程序开发相关技术。
3. 根据设计任务制定工作进度；
4. 完成系统需求分析、概要设计、详细设计及系统测试等工作；

## 总体要求

该系统分为用户（消费者）部分、商家及后台管理部分。

*   **用户（消费者）:** 分为游客和注册用户。游客可以进入主页浏览、了解系统提供商品及服务。
    *   **注册用户的主要功能：** 能够搜索、购买相关商品；加入相关兴趣组、留言并交流；联系商家、享受产品售后服务。
*   **商家部分:** 主要提供游戏、手办等商品的展示、售卖和促销活动；提供与用户的交流平台及售后服务功能。

## 系统功能

### 用户功能模块

1.  **个人中心：**
    *   用户注册
    *   用户登录
    *   修改密码
    *   地址管理（用户新建配送地址以及修改默认配送地址）
    *   收藏夹（用户可以将感兴趣的商品添加到收藏夹中）
2.  **商品模块：**
    *   商品展示（展示各类二次元商品的图片、价格、描述等信息，方便用户浏览和选择）
    *   搜索商品（用户可以通过输入关键字来搜索心仪的商品，对搜索的商品根据价格可进行升序或者降序。）
3.  **购物模块**
4.  **商品分类：** 根据不同类型的商品进行分类展示、可以根据商品数据切换不同的分类。
5.  **人气商品：** 对商品的人气进行排名。
6.  **订单管理：** 用户可以查看自己的订单状态，包括待付款、待发货、已发货等。
7.  **支付与结算：** 支持多种支付方式，如在线支付、货到付款等。
8.  **客服沟通：** 用户可以通过客服与店家之间进行交流。
9.  **会员社区：** 会员根据不同种类商品的畅销程度可根据不同的兴趣爱好进入不同类型的社区交流讨论。

### 商家功能模块

1.  **商品管理模块：** 可以对商品进行管理（比如：展示、促销、新增、下架、商品销售管理等）
2.  **客服模块：** 可以与用户之间进行沟通交流。
3.  **社区管理功能：** 用于交流社区分类的管理
